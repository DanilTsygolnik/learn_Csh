### Ключевые особенности C#

*Тесная связь с ОС Windows** через .NET Framework*

Как и Java, C# использует библиотеки. Ключевой момент: (??) Шарп сам по себе используюет библиотеку .NET Framework, поэтому программы на нем по умолчанию будут компилироваться и запускаться на любой платформе со средой .NET

*Поддержка мультиязыкового программирования.* 

Благодаря библиотекам .NET программы на C# могут использовать код из других ЯП, тогда как в Java такой возможности нет (вся программа должна быть написана на Java). ==(а как в Джаве, и почему не получается ??)==
