## Использование литералов

Литералы можно рассматривать как форму записи, позволяющую *однозначно* выражать значения констант.

|Тип литерала|Примеры|
|:---:|:---:|
|Целочисленные литералы| `2` ; `10` ; `-5`|
|Литералы с плавающей точкой| `1.45` ; `-4.1` ; `2E+25` ; `5E-9`|
|Щестнадцатеричные литералы| `0xFF` ; `0x1a`|
|Строковые литералы| `"3 apples"` ; `"365"`|
|Буквальные строковые литералы|`@"3 apples"` ; `@"365"`|
|Символьные литералы|`'a'` ; '%'|

Поскольку C# поддерживает различные типы данных для записи чисел, может возникнуть неоднозначность в записи значений. Например, из записи `120.25` непонятно, какой тип данных используется в данной константе (`double` или `decimal`,  в частности).

Для устранения неоднозначности введены дополнительные правила записи литералов.

Каждому числовому литералу соответствует некоторый тип данных "по умолчанию":
- для целочисленных литералов - это `int`, `uint`, `long`, `ulong`;
- для вещественных чисел - `double`.

Если нужно использовать значения других типов, используются  *суффиксы*:
- `12L` - литерал типа `long`;
- `100U` - литерал типа `uint`;
- `984375UL` - это литерал типа `ulong`;
- `10.19F` - это литерал типа `float`;
- `9.95М` - это десятичный литерал типа `decimal`.

### Управляющие последовательности

К литералам относят т.н. *управляющие последовательности символов* (escape-последовательности):

|Управляющая последовательность| Описание|
|:---:|:---:|
|`\a`| Звуковой сигнал (звонок)|
|`\b`| Возврат на одну позицию|
|`\f`| Перевод страницы (переход на новую страницу)|
|`\n`| Новая строка (перевод строки)|
|`\r`| Возврат каретки|
|`\t`| Горизонтальная табуляция|
|`\v`| Вертикальная табуляция|
|`\0`| Пустой символ|
|`\'`| Одинарная кавычка|
|`\"`| Двойная кавычка|
|`\\`| Обратная косая черта|

Кроме того, с помощью такой записи можно представить любой символ Unicode. Для этого используется специальный формат `\uhhhh` или `\xhhhh`, где `u` - обязательный префикс, `h` -  шестнадцатеричные цифры. Примеры:
- `\u0066` соответствует символу 'f' ;
- `x2B`соответствует символу 't' .

---

С управляющими последовательностями связана особенность *буквальных литералов* в C#: в их записи, внутри кавычек, набирается текст в привычной для чтения форме, т.е. специальные символы в явном виде не используются. Требуется лишь помнить об оформлении кавычках внутри такой записи (вместо `\"` и `\'` использовать `\"` и `\'`).
