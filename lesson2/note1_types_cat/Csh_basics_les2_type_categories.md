## Введение в систему типов C#

Типы в C# можно разделить на 3 группы:
- предопределенные (встроенные, built-in);
- библиотечные (обычно из стандартной библиотеки .NET Framework);
- определенные программистом (пользовательские).

Как нетрудно заметить, в C# имеется класс **object** (общий базовый тип), от которого наследуются все остальные типы.

*Предопределенные типы всегда включены в язык C#.* Прежде чем перейти к их рассмотрению, стоит разобраться в разнице между типами значениями и типами ссылками.

## Объявление переменных

В соответствии со строгой типизацией C#, всякая переменная должна быть обязательно инициализирована с указанием ее типа.

Формат инициализации переменных:
```
var_type name = expression; 
```
где:
- `var_type` - название типа переменной;
- `name` - имя переменной;
- `expression` - инициализируеющее выражение.

Примеры:
```c#
// инициализация с присваиванием
int apples_num = 50;
double pi=3.141593, r=2.0;
double circle_area = pi*r*r;

// присваивание отдельно от инициализации
double electron_charge; // объявление
electron_charge = -1.602e-19; // присваивание значения
```

В одном объявлении могут быть определены несколько переменных.

В качестве имени переменно недопустимо использование [[Csh_basics_les2_reserved_words|зарезервированных слов]].

При иницилизации переменных, чье значение должно быть некоторой константой, используются *литералы*.

**Литералы** - переменные, которые сами по себе представляют некоторое значение и *не требуют предварительной инициализации*.      
Подробнее об использовании литералов - в [[Csh_basics_les2_literals|заметке]].
